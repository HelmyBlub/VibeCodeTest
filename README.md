# 3D Game Prototype

A browser-based 3D action RPG prototype built with Babylon.js and TypeScript.
Inspired by World of Warcraft — third-person movement, magic casting, and enemy combat.

## Getting Started

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Tech Stack

- [Babylon.js](https://www.babylonjs.com/) — 3D engine
- [TypeScript](https://www.typescriptlang.org/) — language
- [Vite](https://vitejs.dev/) — dev server and bundler
